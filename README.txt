Elements is used to convert Celsius to Kelvin or Kelvin to Celsius.
The user selects the element that they desire to lookup.
The temperature that is entered, is then used to calculate what state is the element at this temperature.

The user input is taken from the first activity and is and is passed as strings to the second activity by using putExtra.
Strings are displayed by using TextViews.
A spinner is used to the select an element.
EditText is used to input the temperature.
RadioButtons are used to specify whether the temperature that is entered is Kelvin or Celsius.
ImageView is used to display the image of the desired element.
